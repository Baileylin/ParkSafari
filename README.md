# ParkSafari
Explore the biodiversity at the destination of your next journey.

## Backend
Before running, create `config.json` with the following values:
```json
{
  "rds_host": ...,
  "rds_port": ...,
  "rds_user": ...,
  "rds_password": ...,
  "rds_db": ...,
  "server_host": ...,
  "server_port": ...
}
```

To start the server, run `npm install` and `npm start`.
